# 祖马消除小游戏

一个纯前端、无后端的 HTML Canvas 祖马消除射击小游戏。

## 运行方式

直接打开 `index.html` 即可游玩。

## 玩法

- 移动鼠标或手指滑动控制炮台方向
- 点击或触摸发射当前彩球
- 点击“切换”交换当前球和备用球
- 3 颗及以上同色珠子相连会自动消除
- 4 颗及以上触发爆炸效果
- 清空轨道珠子通关，珠子进入终点洞口失败
